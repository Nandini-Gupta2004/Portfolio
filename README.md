## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Nandini-Gupta2004/Portfolio.git
```

Go to the project folder

```bash
cd Portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm start
```
